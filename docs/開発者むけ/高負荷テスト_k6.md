# 高負荷テスト（負荷シミュレーション）ガイド：k6 編

このドキュメントでは、開発中の「長田検定」等のWebサイトに対して、
k6を使用した高負荷テストを行う手順を解説します。

---

## 1. k6 とは？
- Go言語製の軽量・高速な負荷テストツール。
- テストシナリオを **JavaScript (ES6)** で記述可能。
- 同時接続数（VUs）や実行時間を細かく制御し、サーバーの限界値を測定します。

---

## 2. インストール手順

### Mac ユーザー
Terminal で以下を実行：
`brew install k6`

### Windows ユーザー
1. [k6公式リリースページ](https://github.com/grafana/k6/releases) から 
   `k6-vX.X.X-amd64.msi` をダウンロードしてインストール。
2. または、PowerShell（管理者）で `winget install k6` を実行。

---

## 3. VS Code での準備
1. 拡張機能 **「k6」** (Grafana製) をインストール（構文ハイライト用）。
2. プロジェクトのルートに `load-test.js` を作成。

---

## 4. テストシナリオの作成 (load-test.js)

```javascript
import http from 'k6/http';
import { sleep, check } from 'k6';

// テストの設定（段階的に負荷を上げる例）
export const options = {
  stages: [
    { duration: '20s', target: 20 }, // 20秒かけて20人まで増やす
    { duration: '20s', target: 50 }, // 次の20秒で50人まで増やす
    { duration: '10s', target: 0 },  // 10秒で終了
  ],
};

export default function loadTest() {
  // ポート転送URL（Public設定済み）またはデプロイ先URL
  const url = 'ここにリンクを貼り付けてね';
  
  const res = http.get(url);

  // 正常性の確認
  check(res, {
    'status is 200': (r) => r.status === 200,
    'has expected text': (r) => r.body.includes('長田'),
  });

  // ユーザーの思考時間をシミュレート（1〜3秒のランダム）
  sleep(Math.random() * 2 + 1);
}
