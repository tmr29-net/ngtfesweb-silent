# その他の追加ページの仕様書
---
## 仕様書の目的
この仕様書は、文スタ紹介ページを除く今後追加される個別のページの仕様を記述するものである。
##  追加ページ
### 1. Access ページ

#### 1.1. 目的
* 利用者に地図を提示しつつ長田高校までの道のりを提示する(Access機能)
* 利用者に文化祭における主に入場に伴う注意事項を知らせる(Cauation機能)

#### 1.2. Access機能

実施機能
* 長田高校のホームページの「学校紹介/アクセス方法・構内図」における「アクセス方法」に記載された地図と詳細な道のりについて記載
* 必要があれば、長田高校の詳細な住所と郵便番号（去年のwebサイトではフッターでの提示あり）
* インラインフレーム等を用いたGoogleMapの埋め込み（必要に応じてGoogleMap自体のリンクへの誘導）
* 前述の機能に加えて、必要があればより詳細なまたは多様なアクセス方法の提示（これはこの機能の実装を強制するものではない）

#### 1.3. Caution機能

実装機能
* 主に一般祭で、招待制の明示や入場方法といった文化祭の入場にかかる機能の提示
* 必要があれば、長田高校にアクセスする上での注意条項（車での往来の可・不可等）
* 前述の機能に加えて、必要があれば文化祭全体あるいは個別の注意事項の提示（これはこの機能の実装を強制するものではない。また、去年のwebサイトにはトップページにこの記載があった。）

### 1.4. ページ上での仕様
* ヘッダーにおいては、このページのリンクを追加する際は、特に理由がなければ最も下位に（右または下）に置くものとする。
* ドメイン名は、特に理由がなければ「access」とする。
