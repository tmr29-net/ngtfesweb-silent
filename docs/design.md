# デザイン　仕様書
この仕様書は、webページの背景及び文字の色、組み合わせ、または文字のサイズ、ウェイト、フォント、配置などを定めるものである。
## 1.文字
文字は以下のように定める。（無視しても良い）
### 1.1.共通
* 行間は1.4より大きいものとする。
* タイトルはブロックの中央寄せとする
* 説明文と見出しは左寄せとる。

フォントサイズにおいては、以下を一例とする
|役割|サイズ|ウェイト|
|----|----|----|
|文化祭テーマ等の大タイトル|72px~|bold|
|メインタイトル|32px~|Bold|
|見出し|24px|Bold|
|本文等（標準）|16px|Regular|
|補足・注釈|12px|Regular|

フォントはすでに使用されているInter, Inter Fallbackを基本として、sans-serifのフォントを用いる。


## 2.色
色については以下のように定める。
* 黒は #222222 (lab(13.228 0 0)) （可読性向上と目の疲れの軽減のため）または #000000 (lab(0 0 0)) を使用する

背景色は、
1. <span style="color:#ffffff"> #ffffff</span> （lab(100 0 0)）
2. <span style="color:#bbefff"> #bbefff</span> (lab(91.233 -14.519 -13.463))
3.  <span style="color:#ccf3ff"> #ccf3ff</span> (lab(93.341 -11.158 -10.213))
4. <span style="color:#ffce6d"> #ffce6d</span> (lab(92.97 -1.401 46.743))
5. 及びその他の色
とする。

可読性に欠ける場合は1を用い、基本的に2と3を交互に用いるものとする。また、問題がなければヘッダー、フッターは1を用いることとする。

文字の色は、背景色と可読性に応じて
1. #222222 (lab(13.228 0 0))または#000000 (lab(0 0 0))
2. #ffffff （lab(100 0 0)）
3. #696969 (lab(44.414 0 0))

等の無彩色を用いる。基本的に1.を使用し、明度が低いときは2.を彩度が高いときは3.を必要に応じて使用することとする。
