- [x] **マイページ: ニックネーム設定機能**
  - [x] 現在、登録フロー以外でニックネームを変更する手段がない。
  - [x] マイページ (`/mypage`) に「プロフィール編集」ボタンを追加し、ニックネームを更新できるようにする。

- [ ] **長田検定: ランキング仕様変更**
  - [ ] **変更**: ランキングは「最高得点」ではなく「今までの合計得点」で作成する。
    - [ ] 併せて「最高得点」とplay_countも表示する。
  - [x] クイズトップページ (`/quiz`) にもランキングを表示する（現在は結果画面のみ）。

- [x] **シードデータの拡充**

  - [x] ※「1カテゴリにつき1票」仕様は維持する。

- [x] **企画一覧: 待ち時間表示 & 画像修正**
  - [x] **新規**: 「推定待ち時間」が表示されていないため実装する。

  - [x] **修正**: 企画一覧の画像が見切れているため、アスペクト比または表示方法 (`object-cover`等) を修正する。

- [x] **トップページ: お知らせ機能**
  - [x] 管理者が設定したお知らせを表示できるようにする。
  - [x] 重要なものは色を変えるなどの強調表示を行う。
