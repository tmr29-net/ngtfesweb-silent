# やること
- クローンする
```bash
git clone https://github.com/NagataFesWeb/2026backupWeb.git
cd 2026backupWeb
npm install
```
作業するときは、
- まず最新の状態にする
```bash
git pull origin main
```
- いきなりmainブランチをさわらず、ブランチを作成し、作業
```bash
git checkout -b ブランチ名
```
